开 adb

短接进 9008，需要一直短接插进去，灯不亮表示成功了

使用 9008 工具刷入 aboot abootbak boot system 分区，刷完重新插拔就可以使用 ARDC 投屏体验安卓 5.1 系统了,同时拥有了 adb

刷 debian

拥有 adb 后，不需要短接进 9008 了，直接命令进

adb reboot edl

用 9008tool 将 debian 包里提供的 aboot 刷到 aboot 分区

之后 format 格式化 boot 分区

此时重新插拔会提示未识别的 usb 设备，需要删除 9008 驱动，删除 adb 驱动，重启电脑再插拔

自动登录校园网

传统艺能，自动登录校园网，我的棒子是直接连的路由器，路由器接的学校的 ac，棒子没有直连校园网，如果你的是直接连的校园网，那就没必要了，连上没啥用

写一个可以自动登录的脚本，直接 curl 登录网站就行了

#!/bin/sh
curl "http://10.2.5.251:801/eportal/?c=Portal&a=login&callback=dr1685877595351&login_method=1&user_account=学号%40运营商&user_password=密码&wlan_user_ip=&wlan_user_mac=000000000000&wlan_ac_ip=&wlan_ac_name=&jsVersion=3.0&_="

学号 密码 运营商需要替换成自己的，运营商用简写

  1. 移动 cmcc
  2. 联通 unicom
  3. 电信 telecom
  4. 校园网 空着不填

将上面脚本的内容保存到/home/user/net.sh 中，直接 echo

echo 'curl "http://10.2.5.251:801/eportal/?c=Portal&a=login&callback=dr1685877595351&login_method=1&user_account=学号%40运营商&user_password=密码&wlan_user_ip=&wlan_user_mac=000000000000&wlan_ac_ip=&wlan_ac_name=&jsVersion=3.0&_="' > /home/user/net.sh

之后安装 crontab 定时任务

apt-get install cron

创建任务

crontab -e

文件最后加一行，每天早上 7：10 自动登录校园网

10 7 * * * /home/user/net.sh

看是否添加成功

crontab -l
